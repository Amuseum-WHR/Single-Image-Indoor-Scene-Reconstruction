# Single Image Indoor Scene Reconstruction
## Requirements
#### requirements if you just want to run a demo：
+ vtk
+ jellyfish
+ seaborn
+ pymesh (unnecessary if you use vtk)
#### requirements if you want to train a demo：
+ python 3.7
+ cuda 11.3
+ pytorch 1.12.1
+ torchvision 0.13.1
+ pytorch3d
+ tensorboard
